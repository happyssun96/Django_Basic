# 🎯 Django_Basic
- 장고를 활용한 블로그 만들기 연습
- 개발환경
  - OS : Windows10
  - IDE : VSCODE
  - HTML, CSS, Python, Django

# 📝 How To Use
```
1. python -m venv myvenv
2. source myvenv/scripts/activate
3. pip install django
4. python manage.py makemigrations
5. python manage.py migrate
6. python manage.py runserver
```
